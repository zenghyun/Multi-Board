## 💻 프로젝트 소개

**Multi Board**는 MongoDB와 Koa를 이용한 백엔드 개발과 서버를 기반으로 리액트를 사용하여 만든 블로그 웹 애플리케이션으로, REST API를 통하여 백엔드와 프론트 엔드 사이에 데이터 교류에 초점을 맞춰서 진행된 프로젝트 입니다.

## 🏷️ 시작 가이드 

### Requirements
For building and running the application you need: 
- Node.js v16.14.0

<br>

### Installation

```js
$ git clone https://github.com/zenghyun/Multi-Board.git
$ cd Multi-Board
```
<br>

### Backend 
```js
$ cd backend
$ yarn install
$ yarn start 
```
<br>

### Frontend
```js
$ cd frontend
$ yarn install
$ yarn start 
```

## ⭐️ 주요 기능

<br>

### 📌 회원가입 및 로그인 기능 
- REST API를 통한 MongoDB와의 정보 공유 
- 중복 이름 체크, 비밀번호 & 비밀번호 확인 동일해야 회원 가입 가능 

<br>

### 📌 게시글 작성 기능 
- 로그인 해야 작성 가능 
- 웹 기반의 텍스트 편집기인 Quill을 이용하여 텍스트를 편집하고 서식 지정 가능 

<br>

### 📌 게시글 수정 및 삭제 기능 
- 내가 작성한 게시글만 수정 및 삭제 가능 

<br>

### 📌 아이디를 통하여 해당 유저가 작성한 게시글만 보는 기능 
- 게시글 리스트에서 아이디 클릭 시 해당 유저가 작성한 글만 보여줌 

<br>

### 📌 태그를 통하여 해당 태그가 적힌 게시글만 보는 기능 
- 게시글 리스트에서 태그 클릭 시 해당 태그가 포함된 게시글만 보여줌
